# Floom v0 Design Handoff

Use `Floom v0 landing.html` as the source of truth.

Do not use `Floom old landing.html` or the older auto-sync positioning.

## Current Product State

Floom v0 lets users download a starter skills pack for their AI agent.

Auto-sync is not ready yet, so it should not be presented as the core launch
promise. It can be mentioned only as a later roadmap idea.

## Keep

- The "Upgrade your agent" value proposition.
- The SkillsBench proof chart.
- The "Use the best skill for the task" section.
- The supported agents strip.
- The bold Swiss/SaaS visual language.

## Primary CTA

Download skills pack.

The linked asset is:

`assets/floom-starter-skills.zip`

## Design Goal

Make the page feel like a polished v0 launch page for agent skills:

- clear download-first flow
- no dependency on account sync
- credible open-source feel
- strong visual proof that skills improve agent output
- easy path for users to understand what they get in the pack

## Suggested Iteration Prompt

Improve this landing page for a v0 launch where the only ready action is
downloading a starter skills pack. Preserve the upgrade-your-agent value prop
and the find-the-right-skill section, but remove or de-emphasize any promise
that Floom automatically syncs skills across agents today.
