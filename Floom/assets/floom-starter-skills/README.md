# Floom Starter Skills Pack

This pack gives an agent three reusable skills to install locally today.

## Included Skills

- `brand-voice`: keep writing consistent with a chosen voice.
- `pr-review`: review pull requests with a focused engineering checklist.
- `startup-financial-model`: build clear first-pass financial models.

## Install

Copy the `skills/` folder into the skills directory for your agent, then restart
or reload the agent so it can discover the new skills.
