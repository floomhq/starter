Quick answers
What is a Floom skill?

A reusable skill package your agents can find, install, and use across Claude, Codex, and more.

How do skills improve my agent?

Skills give your agent focused procedures, examples, and checks for a task. In SkillsBench, curated skills helped agents complete more tasks than running without skills.

What happens when I switch agents?

Your skills stay in the Floom Library, so the next agent can find and use the same setup instead of starting from scratch.

Does it work with my agent?

Yes. Floom keeps skills portable so Claude, Codex, Cursor, Kimi, Gemini, OpenCode, and future agents can work from the same shared library.
