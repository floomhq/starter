About Floom

Curated skills, open source, built in San Francisco.

Floom is open source under MIT. The code lives at github.com/floomhq/starter.

The Starter Pack is 65 hand-picked AI agent skills from skills.sh, the public skill registry. They cover the core surface area an agent needs: code review, design feedback, planning, debugging, copywriting, research, ops.

Why curated: skills.sh has 91,000 plus skills. Most agents drown in choice. We picked the 65 that move the needle, ranked by real install counts, organized into 11 profiles: Core, Dev, Writing, Research, Marketing, Sales, Ops, Founder, Data, Design, Video.

One install. Auto-activates. Works with Claude Code, Codex, Cursor, Kimi, OpenCode.

Built and curated by Federico de Ponte and Adam Beaudoin.

The image: Golden Gate Bridge from the Presidio. Where the work started.
