Privacy Policy

Effective date: 2026-05-11 · Version 1.0

What we collect

The Floom Starter Pack is an open-source CLI tool. When you run the installer:

  • We fetch the public manifest from GitHub (no authentication required)
  • We write skill files to your local project directory
  • We do not collect or transmit your code, project files, or personal information

Analytics

The CLI tool does not include analytics or telemetry. The public manifest is fetched from a GitHub raw URL, which may log standard server access logs (IP address, timestamp, user agent) as per GitHub's own privacy policy.

Website

This website (floom-wireframes.vercel.app) may use Vercel's built-in analytics for basic page view counts. We do not use third-party ad trackers, session recording tools, or fingerprinting.

Contact

Questions? Email fede@rocketlist.ai or open an issue on GitHub.
