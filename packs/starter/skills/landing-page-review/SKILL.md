# Landing Page Review

Use this skill when reviewing a homepage, product page, waitlist page, or launch page.

## Review Order

1. First-screen clarity.
2. Audience and promise.
3. Proof and trust.
4. Primary action.
5. Objections answered.
6. Mobile layout risks.

Give fixes in priority order. Avoid generic marketing advice.
