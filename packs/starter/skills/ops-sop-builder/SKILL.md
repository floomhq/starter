# Ops SOP Builder

Use this skill when documenting a recurring operational process.

## SOP Format

- Purpose
- Trigger
- Inputs
- Steps
- Checks
- Owner
- Escalation
- Done state

Keep steps executable. Include failure cases where the process commonly breaks.
