# Visual QA Review

Use this skill when checking screenshots, web pages, designs, or implemented UI.

## Review Order

1. Broken layout or overlap.
2. Text contrast and readability.
3. Spacing and alignment.
4. Hierarchy and scan path.
5. Button and form affordances.
6. Mobile risks.

Report issues with location, severity, and a concrete fix.
